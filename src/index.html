Projects

Power Selling Program

I participated in a hackathon where I developed software to sell surplus power stored in batteries back to the power grid, effectively utilizing unused solar energy. I solved the issue of unacknowledged power consumption by designing and implementing an object-oriented software solution capable of analyzing extensive datasets. To enhance collaboration and project management, I created a detailed README.md file outlining the project's goals, approach, and feasibility. This initiative is expected to reduce renewable power wastage by approximately 25%, enabling greater solar power utilization and generating profits for both customers and power companies.

twitter bot

The Python-based Twitter bot I developed enables automated tweeting, retweeting, and direct messaging functionalities. To ensure comprehensive functionality, I thoroughly studied the extensive documentation provided by Twitter, allowing me to understand the API's capabilities and limitations. By implementing natural language processing techniques, the bot can analyze tweets and generate appropriate responses. To prevent redundancy and avoid overwhelming Twitter's API, the bot automatically maintains a JSON file that tracks the IDs of tweets it has already interacted with, ensuring it does not repeat actions. This efficient approach optimizes the bot's performance and enhances its overall functionality.

Zombie Survival Game

I developed a zombie survival game using Unreal Engine 5 (UE5) for a game jam. The game features immersive gameplay and challenges players to survive against hordes of zombies. I utilized the advanced features and capabilities of UE5 to create stunning visuals and realistic physics. I implemented a working UI for the typical first person shooters' elements (i.g. Health). The game jam provided a limited time frame to complete the project, and I successfully delivered a fully playable and engaging experience within that timeframe.
